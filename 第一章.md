-  算法概念 ： 特定问题求解步骤的描述，有限的指令集合。

  1. 有穷性
  2. 输入
  3. 输出
  4. 正确性
  5. 可行性

- 算法效率度量 

  > 和问题规模有关

  1. 时间复杂度：

     - 频度；通常用n表示
      
       ```c
       int n=3;
           for (int i = 0; i < n; i++)
           {
           puts("output");
           }
           return 0;
       ```

       频度=n

       C(n)=3n+1
       
       f(n)=3n                f(n)基本运算的频度 （最深层循环内的语句）
       
       T(n)=O(f(n))=O(n)   O取数量级运算为
       
     - 频度之和：C(n)
     
     - 时间复杂度: T(n)
